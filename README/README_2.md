## ⚙️ 실행 방법
<!-- Java 파일 또는 Spring 프로젝트 실행 방법을 작성해 주세요 -->
step1
    사용자로부터 이름, 전공, 기수를 입력받고 입력값을 검증한 후 Lion 객체를 생성
step2
    입력값을 바로 Lion 객체로 전달하고, 검증
step3
    접근제어자에 따라 내부 클래스 접근을 시도하며 접근 가능 여부를 확인
## 💡 작업 내용
- 구현한 기능 목록
    Lion 클래스를 정의하여 아기사자 한 명의 정보를 객체로 표현
    생성자를 통해 객체 생성 시 초기값을 설정
    validate() 메서드를 구현하여 객체 내부에서 상태 검증 수행
    step1에서는 main에서 입력값 검증을 수행하여 책임 분리 구조 설계
    step2에서는 검증 책임을 Lion 클래스 내부로 이동시켜 객체 중심 설계
